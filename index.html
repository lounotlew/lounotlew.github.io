BlizzPy: Open-Source Blizzard API Wrapper Library

BlizzPy is an open-source Blizzard API wrapper library for Python. I built the library from scratch to fill the lack of easy, comprehensive access to Blizzard’s API for Python programmers. BlizzPy returns both raw API JSON data straight from the API and clean, organized data pre-processed within the library, and includes various analytical functions to help develop insights into the data. These features give users the options to explore the raw game data themselves, or jump straight into working with pre-processed data so they can easily build whatever it is they need to build.

The project is published on PyPI, and we're always looking for new contributors.

Sentweetment2: Twitter Sentiment Analyzer

Sentweetment2 is a Django web application designed for performing sentiment analysis on Twitter users. All you need to do is type in a Twitter username, and the web app will take care of everything. You will see a general overview of the selected user's account activity and their tweets' polarities, and can optionally view more detailed statistics and visualizations through Highcharts.

Check out Sentweetment2 here.

All Your Restaurant Are Belong To Us: Interactive Yelp Data Downloader

All Your Restaurant Are Belong To Us (AYRABTS) is a R Shiny web application for interactively downloading Yelp API data. Instead of relying on script files to connect to and download data from the Yelp API, which can get confusing, AYRABTS takes in your desired parameters and visualizes a data frame in real-time so you know exactly what kind of data you're getting. You can then download the data frame as a .csv straight to your computer.

Check out AYRABTS here.

Networkify: Build Your Own Networks and Systems

Networkify is a Java-based light desktop app I'm currently working on. The idea behind it is to use build directed/undirected weighted graphs with an intuitive UI to represent various networks and systems, such as transportation networks or a recommendation system. You can then analyze your network with well-known algorithms such as Dijkstra's or Bellman-Ford with a simple button. An example use case of Networkify would be to create a delivery route network, and find the fastest route between a restaurant and a customer.

FileCase: Lossless File Compressor and Encrypter

FileCase is a Python-based desktop app for lossless file compression and encryption of .txt and .docx files. It uses a highly organized internal file structure to maintain data persistence, so all you have to worry about is clicking compress/decompress (or encrypt/decrypt) and keeping track of your file. FileCase reduces sizes by 50% for .txt files, and up to 67% for .docx files. It also uses AES-256 CFB for encryption.

Gitlet: Mini Version-Control System

Gitlet is a mini version-control system for local directories on your machine that mimics most standard features of Git, such as commits, logs, and merges. Gitlet uses SHA-1 hashing to reference different commits (for pulls and merges), and utilizes Java Serialization to read and write file content to a hidden backup directory. I developed Gitlet with another fellow student as a part of CS 61B's final project.
